# pangolin-eval v0.2.1

`v0.2.1` is a maintenance release for GitHub Actions runtime compatibility.

## Highlights

- Updated pinned `actions/checkout` and `actions/setup-python` SHAs to versions
  whose action metadata uses `node24`.
- Kept CI and release workflows pinned by commit SHA.

## Compatibility

No runtime or schema changes are expected from `v0.2.0`.

## Verification

Release readiness should include:

```bash
PYTHONPATH=src python3 -m unittest discover -s tests
find schemas examples -name '*.json' -print0 | xargs -0 -n1 python3 -m json.tool >/dev/null
PYTHONPATH=src python3 -m pangolin_eval.cli validate --config examples/ci_gate/config.json
```

GitHub Actions should pass without Node 20 deprecation warnings.
