# pangolin-eval v0.2.3

`v0.2.3` is a pre-publication hardening release for safer public sharing. It does not introduce breaking schema or CLI changes.

## Added

- Public `SECURITY.md` with a private vulnerability reporting path and sensitive-data guidance.

## Changed

- Live-provider README examples now default to `metadata-only` report content.
- Release and CI package build commands pin `build`.
- Package version updated to `0.2.3`.
- Launch notes avoid early Pro-specific roadmap wording.
- Compatibility and migration notes point at the current `v0.2.x` release line.

## Security And Safety

- Escaped user-controlled Markdown table cells in comparison, RAG, and TraceCard reports.

## Compatibility

No breaking changes are expected from `v0.2.2`.

Report consumers should continue branching on `schema_version` rather than package version.
